##  前言
✅本项目实现一个信息交流平台，主要包括提问、讨论等形式对一些问题，创意进行分享。

## ⬆ 项目名称：PopIdea
### 1 项目介绍
- ✅首先，PopIdea的定位是一个知识分享的社区，有问题、文章、点子三个个主要的概念。用来进行内容的一个分类以及传播。
